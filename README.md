# SattvikAI

SattvikAI ek AI-powered food analysis aur healthy eating assistant hai.

## Features
- User Login & Signup
- Food Logging
- AI Analysis
- Streak Tracking
- Database Support

## Tech Stack
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Run Project

1. Clone repo
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py
